# Vulandar

Here is a tiny calendar app made with Vue.js, using [Moment.js](https://momentjs.com/)

Includes:
* Month by month navigation
* Event creation
* Event view
* Event removal

But also:
* Yearly recurring events
* Colors for each event
* Supports light and dark browser prefered theme
* Supports mobile and desktop resolutions
* Shortcut button to back to current month
* Automated deployment to GitHub pages at [https://arannou.github.io/Vulandar/](https://arannou.github.io/Vulandar/)

Ideas to go further:
* Support internationalization EN | FR and set monday to start the week
* Add week number
* Add persistency 

## To install

* Clone this repo
* Ensure to have Node.js LTS installed
* Run `npm install`
* Run `npm run dev` and open the browser with Vite default port 5173
